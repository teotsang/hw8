What is an SQL versus a noSQL database? 

SQL is a relational database and noSQL is a no-relational database
SQL data is presented in table form and noSQL is presented with documents, key-value pairs or graphs.
SQL uses structured query language to do changes on data and noSQL uses unstructured query language which use on collection of data.

Why is one preferable to another? 

SQL is preferable than noSQL when data are in same categories such as storing account informations, usernames and passwords.
NoSQL is preferable than SQL when there are lots of media showing in one page and users can interact with them frequently such as a Facebook page.
